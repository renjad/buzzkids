# Buzzkids English Club - Student Management System

## About

Web application for managing students, lesson bookings, and learning content for an English school.

## Features

- Student & booking management
- Subscription & ticket-based plans
- Learning content library (PDFs & YouTube)
- LINE Official Account notifications

## Requirements

- PHP 8.2+
- Composer
- Docker & Docker Compose (for Laravel Sail)
- MySQL 8.0
- Node.js & NPM
